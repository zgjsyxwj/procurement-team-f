## 已拆分流程

1. 供应商创建与邀请
2. 创建询价单
3. 供应商报价
4. 采购员代录报价
5. 核价与审批
6. 合同创建、OA审批与签署
7. PO履约与付款闭环
8. 数据设置：字段库与表单配置

## 推荐演示顺序

供应商创建与邀请 → 创建询价单 → 供应商报价/采购员代录报价 → 核价与审批 → 合同创建、OA审批与签署 → PO履约与付款闭环。
